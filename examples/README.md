# Beispiel-Daten

`beispiel-sammlung.xlsx` ist eine **fiktive** Modellauto-Sammlung (18 Einträge,
u. a. Wiking, Siku, Majorette, Herpa, Schuco, Brekina) im gleichen Spalten-Format
wie eine echte Sammler-Excel.

**Zweck:** die App ohne eigene Daten ausprobieren. In der laufenden App oben auf
**„Import"** klicken und diese Datei hochladen — danach zeigen Galerie, Statistik,
Lücken und Wunschliste sofort Beispiel-Inhalte.

Alle Nummern, Preise und Zustände sind erfunden und dienen nur der Demonstration.
